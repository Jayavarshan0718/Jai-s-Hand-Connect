# Jai-s-Hand-Connect
An AI-powered hand tracking web application built with MediaPipe Hands, JavaScript, and HTML5 that detects real-time hand gestures and creates interactive AR visual effects, particles, lighting animations, and audio feedback using webcam input.

# Hand Connect – AI Hand Tracking AR Experience

## Overview

Hand Connect is an AI-powered web application that demonstrates real-time hand tracking and gesture recognition using Google's MediaPipe Hands framework. The application captures live webcam input, detects hand landmarks, and generates interactive augmented reality (AR) visual effects, animations, and audio feedback based on user gestures.

The project showcases the integration of computer vision, web technologies, and interactive graphics to create a touchless human-computer interaction experience.

---

## Features

- Real-time hand tracking using MediaPipe Hands
- Detection of up to two hands simultaneously
- Tracking of 21 landmarks for each detected hand
- Gesture recognition including:
  - Pinch
  - Open Hand
  - Fist
- Interactive particle effects
- Dynamic lightning effects between hands
- Shockwave animation on gesture detection
- Matrix-inspired animated background
- Multiple visual themes
  - Rainbow
  - Cyberpunk
  - Lava
  - Ocean
  - Galaxy
- Dynamic audio effects using the Web Audio API
- Performance dashboard displaying:
  - Hands detected
  - Frames per second (FPS)
  - Current gesture
  - Hand spread percentage

## How It Works

1. The user opens the application in a web browser.
2. Camera permission is requested.
3. MediaPipe Hands detects and tracks the user's hands in real time.
4. Hand landmarks are processed to recognize gestures.
5. Based on the detected gestures, the application renders interactive visual effects and generates audio feedback.
6. Users can switch between multiple visual themes during runtime.
